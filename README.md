SMAF形式の着メロ・音色ライブラリのパーサです。

gopkg.in 経由にてバージョン指定でインポートしてください。

```go
import "github.com/but80/go-smaf/v2"
```

使用例は [smaf825のv2ブランチ](https://github.com/but80/smaf825/tree/v2) のソースをご覧ください。
