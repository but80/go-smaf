SMAF形式の着メロ・音色ライブラリのパーサです。

gopkg.in 経由にてバージョン指定でインポートしてください。

```go
import "gopkg.in/but80/go-smaf.v1"
```

使用例は [smaf825](https://github.com/but80/smaf825) のソースをご覧ください。
